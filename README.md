# Cashflow Forecaster

This is a cashflow-forecasting app. Built in Vue 3 using Composition API / TypeScript, using Pinia
for local state management and Cypress to ensure the app works as expected.

Environment: MacOS Tahoe 26.4.1

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
